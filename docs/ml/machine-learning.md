# 机器学习基础学习笔记

## 什么是机器学习

机器学习(Machine Learning)是人工智能的一个分支，它使计算机系统能够从数据中学习并做出决策或预测，而无需被明确编程来执行特定任务。它是让计算机具备"学习"能力的科学。

## 机器学习发展历程

### 1. 早期阶段(1950s-1970s)
- 1950年：艾伦·图灵提出图灵测试
- 1957年：感知机算法诞生
- 1967年：最近邻算法提出

### 2. 发展阶段(1980s-1990s)
- 1980年：决策树算法ID3提出
- 1986年：反向传播算法广泛应用
- 1995年：支持向量机(SVM)提出

### 3. 深度学习兴起(2000s-至今)
- 2006年：深度学习概念提出
- 2012年：AlexNet在ImageNet竞赛中获胜
- 2016年：AlphaGo战胜围棋世界冠军

## 机器学习基本概念

### 1. 数据集
- **特征(Features)**：描述样本的属性
- **标签(Label)**：样本的目标输出
- **训练集**：用于训练模型的数据
- **测试集**：用于评估模型性能的数据

### 2. 模型
- **假设空间**：所有可能的模型集合
- **损失函数**：衡量模型预测与真实值的差距
- **优化算法**：寻找最优模型参数的方法

### 3. 评估指标
- **准确率**：正确预测的样本占总样本的比例
- **精确率**：预测为正例中实际为正例的比例
- **召回率**：实际为正例中被正确预测的比例
- **F1分数**：精确率和召回率的调和平均

## 机器学习主要类型

### 1. 监督学习(Supervised Learning)
使用带有标签的数据进行训练

#### 主要任务：
- **分类**：预测离散类别标签
- **回归**：预测连续数值

#### 常见算法：
- 线性回归
- 逻辑回归
- 决策树
- 随机森林
- 支持向量机
- 朴素贝叶斯

### 2. 无监督学习(Unsupervised Learning)
使用无标签数据进行训练

#### 主要任务：
- **聚类**：发现数据中的群体结构
- **降维**：减少特征数量
- **异常检测**：识别异常数据点

#### 常见算法：
- K-means聚类
- 层次聚类
- 主成分分析(PCA)
- 自编码器

### 3. 强化学习(Reinforcement Learning)
通过与环境交互学习最优策略

#### 核心要素：
- **智能体(Agent)**：学习和决策的主体
- **环境(Environment)**：智能体交互的外部世界
- **状态(State)**：环境的描述
- **动作(Action)**：智能体的选择
- **奖励(Reward)**：环境对动作的反馈

### 4. 半监督学习
结合少量标记数据和大量未标记数据进行学习

### 5. 自监督学习
从未标记数据中自动生成监督信号进行学习

## 常见机器学习算法

### 1. 线性回归
用于预测连续数值，假设特征与目标之间存在线性关系

数学表达式：
```
y = w₀ + w₁x₁ + w₂x₂ + ... + wₙxₙ
```

### 2. 逻辑回归
用于二分类问题，通过sigmoid函数将线性回归输出映射到概率

数学表达式：
```
P(y=1|x) = 1 / (1 + e^-(w₀ + w₁x₁ + ... + wₙxₙ))
```

### 3. 决策树
通过树状结构进行决策，每个内部节点表示特征判断，叶节点表示类别

优点：
- 易于理解和解释
- 不需要数据预处理
- 能处理数值型和类别型数据

缺点：
- 容易过拟合
- 对噪声敏感

### 4. 随机森林
多个决策树的集成方法，通过投票或平均得出最终结果

优点：
- 减少过拟合
- 处理缺失值能力强
- 评估特征重要性

### 5. 支持向量机(SVM)
寻找最优超平面将不同类别的数据分开

核心思想：
- 最大化类别间的间隔
- 使用核函数处理非线性问题

### 6. K-means聚类
将数据划分为K个簇，使簇内数据相似度高，簇间相似度低

算法步骤：
1. 随机初始化K个聚类中心
2. 将每个数据点分配给最近的聚类中心
3. 更新聚类中心为簇内所有点的均值
4. 重复步骤2-3直到收敛

## 特征工程

### 1. 特征选择
- 过滤法：基于统计指标选择特征
- 包装法：通过模型性能选择特征
- 嵌入法：在模型训练过程中进行特征选择

### 2. 特征提取
- 主成分分析(PCA)
- 线性判别分析(LDA)
- 独立成分分析(ICA)

### 3. 特征构造
- 多项式特征
- 交互特征
- 分箱处理

### 4. 特征缩放
- 标准化：均值为0，方差为1
- 归一化：缩放到[0,1]区间
- 最大绝对值缩放

## 模型评估与验证

### 1. 交叉验证
- K折交叉验证
- 留一交叉验证
- 分层交叉验证

### 2. 过拟合与欠拟合
- **过拟合**：模型在训练集上表现好，测试集上表现差
- **欠拟合**：模型在训练集和测试集上表现都不好

### 3. 正则化
- L1正则化：Lasso回归
- L2正则化：Ridge回归
- Elastic Net：L1和L2的组合
